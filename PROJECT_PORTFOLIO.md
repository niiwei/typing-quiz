## 核心功能

### 1. 强制主动输出 - "单框连击"交互

- **创新交互设计**：无需手动提交，输入即匹配
- **防自我欺骗**：只有敲对关键词，答案才会砰然跳出
- **无序答题**：任意顺序回答，无需死记硬背顺序

### 2. 科学间隔复习 - 基于 Anki SM-2 算法

- **5种状态流转**：新测验 → 学习中 → 待复习 → 今日复习 → 重学中
- **4级评级体系**：重来/困难/良好/简单，算法自动计算下次复习时间
- **遗忘曲线对抗**：在遗忘临界点精准推送复习

### 3. 完整用户系统

- JWT 无状态认证（24小时有效期）
- BCrypt 密码加密
- 严格账户数据隔离（所有查询按 userId 过滤）

### 4. 测验管理体系

- 测验分组管理（多对多关联）
- JSON 导入导出
- 12个官方测验包（新用户注册自动获取）

## 技术亮点

### 后端架构

```
Controller → Service → Repository → Entity
     ↓            ↓              ↓
  REST API    业务逻辑      Spring Data JPA
```

- **框架**：Spring Boot 2.7.18
- **数据访问**：Spring Data JPA + Hibernate
- **数据库**：MySQL 8.0，支持级联删除、多表关联
- **安全**：JWT (JJWT) + Spring Security (BCrypt)
- **部署**：Docker + 阿里云服务器 (47.102.147.127)

### 核心算法实现

**SM-2 算法核心逻辑：**
```java
// 新间隔 = 旧间隔 × 简易度系数 × 评级乘数
int newInterval = (int) (oldInterval * (easeFactor / 1000.0) * ratingMultiplier);

// 简易度动态调整：评级好则↑，差则↓（范围 1300-5000）
int newEaseFactor = easeFactor + (rating == 4 ? 150 : (rating == 1 ? -200 : 0));
```

### 数据库设计

| 核心表 | 说明 |
|--------|------|
| `users` | 用户表（BCrypt密码） |
| `quiz` | 测验主表（类型枚举、账户隔离） |
| `answer` | 答案表（标准化内容支持大小写不敏感） |
| `quiz_group` | 分组表（多对多关联） |
| `quiz_review_status` | 复习状态表（SM-2算法核心数据） |
