# 数据库管理功能说明

## 功能概述

数据库管理页面提供了强大的数据查询和检索功能,支持按测验维度和答案维度进行双向检索。

## 访问地址

http://localhost:8080/database.html

## 主要功能

### 1. 数据库统计

页面顶部显示实时统计信息:
- 测验总数
- 答案总数

### 2. 按测验维度检索

支持两种搜索方式:

#### 按测验ID搜索
- 输入测验ID(纯数字)
- 显示测验详细信息
- 显示该测验的所有答案列表

#### 按测验名称搜索
- 输入测验名称关键字(支持模糊搜索,忽略大小写)
- 显示所有匹配的测验列表
- 点击"查看答案"可查看该测验的所有答案

### 3. 按答案维度检索

支持两种搜索方式:

#### 按答案ID搜索
- 输入答案ID(纯数字)
- 显示答案详细信息
- 显示该答案所属的测验信息

#### 按答案内容搜索
- 输入答案内容关键字(支持模糊搜索,忽略大小写)
- 显示所有匹配的答案列表
- 显示每个答案所属的测验信息
- 点击"查看详情"可查看答案和测验的详细关联

## 使用场景

### 场景1: 查看某个测验的所有答案
1. 切换到"按测验检索"标签
2. 输入测验ID或名称
3. 点击"搜索"
4. 查看测验信息和答案列表

### 场景2: 查找某个答案属于哪个测验
1. 切换到"按答案检索"标签
2. 输入答案ID或内容
3. 点击"搜索"
4. 查看答案所属的测验信息

### 场景3: 搜索包含特定关键字的所有答案
1. 切换到"按答案检索"标签
2. 输入关键字(如"北京")
3. 点击"搜索"
4. 查看所有包含该关键字的答案及其所属测验

### 场景4: 搜索名称包含特定关键字的所有测验
1. 切换到"按测验检索"标签
2. 输入关键字(如"首都")
3. 点击"搜索"
4. 查看所有匹配的测验列表

## API端点

### 获取数据库统计信息
```
GET /api/database/stats
```

### 按测验ID查询答案
```
GET /api/database/quiz/{id}/answers
```

### 按测验名称搜索
```
GET /api/database/quiz/search?name=关键字
```

### 按答案ID查询所属测验
```
GET /api/database/answer/{id}/quiz
```

### 按答案内容搜索
```
GET /api/database/answer/search?content=关键字
```

## 技术特点

- **智能识别**: 自动识别输入是ID(纯数字)还是关键字
- **模糊搜索**: 支持关键字模糊匹配,忽略大小写
- **双向检索**: 支持从测验查答案,也支持从答案查测验
- **实时统计**: 显示数据库实时统计信息
- **友好界面**: 清晰的表格展示,支持快速导航

## 注意事项

1. 搜索关键字区分ID和内容:纯数字会被识别为ID搜索
2. 关键字搜索支持部分匹配,不需要输入完整内容
3. 搜索结果按相关度排序
4. 点击搜索结果中的链接可以快速查看详细信息
